Audience and Goals

Usually these types of games are targeted towards younger girls but I want this collection of games to be enjoyed by all ages. My main audience for this project are my friends, classmates and professor. What I wanted to accomplish with these games was drag and drop interactivity using graphics I created. I also wanted there to be separate sections that the user can toggle between to use different types of clothing.

My Process and Problems

Originally I had attempted to use PIXI.JS to add the drag and drop functionality to my project, but I had a lot of problems getting it to work. It took me a couple days to even get the library to link to my files properly, and when I was able to get the drag and drop functionality working, the sprites were not able to be incorporated into my grid and would load in one place on the screen even when the browser was resized. After this I tried to use D3.JS but also ran into a few roadblocks. I was, again, able to get the drag and drop functionality working but the images were only visible in the parent container and could not be moved onto the other parts of the canvas.
